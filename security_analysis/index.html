安全分析
项目简介：安全分析是指对网络系统、数据等进行实时监控和分析，以发现和应对各种网络威胁和攻击的过程。通过可视化技术，安全人员可以将网络数据流量和安全事件转化为直观的图表和报告，有助于更好地识别网络威胁和攻击来源，以制定更加有效的安全策略和措施。FDU-VIS专注于对网络安全状态的可视分析，帮助安全人员更好地理解和分析网络数据流量和安全事件。例如，我们使用网络分析技术定位网络资产，并使用可视化方法监控其实时状态。我们还监控以太坊网络状态并对其上的安全风险进行评估。
Project Introduction: Security analysis refers to the process of real-time monitoring and analysis of network systems and data to detect and respond to various network threats and attacks. Through visualization technology, security personnel can transform network data traffic and security events into intuitive charts and reports, which help identify the sources of network threats and attacks more effectively and develop more efficient security strategies and measures. To help security personnel comprehend and analyze network data traffic and security events, FDU-VIS focuses on visual analysis of network security status. For instance, we locate network assets using network analysis technology and utilize visualization techniques to track their real-time status. We also monitor and report the Ethereum blockchain's health and assess its security risks.
